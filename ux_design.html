Experience Design

Competitive Benchmark

The objective of this project was to create a new flight booking experience. I started by performing a competitive analysis in order to see what kind of flight booking experiences were currently being used by popular airlines.

Usability Testing

I then selected two airlines and used their apps and ran a 1 hour usability test with a user. I gave them a specific scenario where they needed to book two tickets, to a specific destination, around a specific week of the month.

Affinity Mapping

After running the usability test I gathered all the data from the test and the competitive benchmark and broke it down into individual notes. I then posted them on the wall and, similar to unsupervised learning, worked to see what themes naturally emerged.

User Journey Mapping

I then placed these themes into a user journey map, in order to better understand the current user model. I used this map to identify points in the journey where I could target in order to solve issues, or deliver more value to the user.

User Flow

After working out a design, I broke it down into a detailed user flow. In the user flow I clarified each and every possible interaction and screen state for the design.

Wireframe

I then converted the interactions into wireframes with hand drawings. Drawing by hand allowed me to think more slowly about the information architecture and different things that should be included in the app.

Drawing

Hand drawing can be time consuming but really does help lead to amazing ideas. I am planning to actually order some stencils for the future to make the process faster.

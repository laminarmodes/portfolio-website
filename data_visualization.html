Data Visualization

Dashboard design in Adobe XD

Information visualization and dashboard design has been a very important area of UX for digital products. This first dashboard explores the use of monochromatic colors and distinct and specific shapes to develop a consistent language and make the data easy to interpret.

Online Banking UI design in Figma

The next visualisation is a responsive UI for an online banking app designed in Figma. The theme here is fun, trustworthy and playful. The form structure of the logo represents a purple coral reef and makes up the primary color of the UI design. The inner parts of the logo represent colorful fish living inside and this is seen in the secondary colors that represent different types of accounts.

iPhone

There is quite a lot of information to view so rather than cramming everything onto a single screen, the user is encouraged to scroll horizontally to see month-to-month information.

iPad

The iPad navigation can be the same as the iPhone or alternatively placed to the right or left side for easier reaching while holding the iPad.

Desktop

The desktop design is almost exactly the same as mobile, to ensure that the user will not get confused when switching devices. The same SwiftUI codebase could be used for all iOS platforms and the navigation can be implemented to automatically adjust to the device within the same parts of the code.
